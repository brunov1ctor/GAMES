# Eventos Dinâmicos nas Quests

## Intenção

Misturar eventos dinâmicos com missões. Quests nunca são experiências estáticas.

---

## Funcionamento

Durante qualquer quest podem ocorrer eventos aleatórios:

| Evento | Tipo |
|--------|------|
| Ataques de bandidos | Combate |
| Tempestades | Ambiental |
| Terremotos | Ambiental |
| Avalanches | Ambiental |
| Chefes raros | Combate |
| Portais | Exploração |
| Robôs antigos | Combate/Exploração |
| Desabamentos | Ambiental |

Os eventos alteram **objetivos, recompensas e inimigos** em tempo real.

---

## Integração com Estado Mundial

O estado atual do mundo modifica as quests.

### Exemplo: Se a Floresta Negra existir

- Mais monstros
- Neblina permanente
- Novos NPCs
- Novos chefes
- Novas recompensas
